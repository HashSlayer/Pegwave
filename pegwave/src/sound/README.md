# UI sound effects

Menu and overlay SFX (button clicks, round win, blind select). Separate from collision synth in `../audio/`.

- `soundEffects.ts` — effect IDs and placeholder asset URLs
- `useSoundEffects.tsx` — play/mute/volume
- `SoundContext.tsx` — React provider (wraps app in `App.tsx`)
- `SoundSettings.tsx` — settings modal
