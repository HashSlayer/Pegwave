# Pegwave — Provenance & License Notice

## Project

**Pegwave** is an original Plinko / synth roguelike. Gameplay, audio systems, and
branding are Pegwave-specific.

## Scaffold

Early run-loop and roguelike scaffolding was adapted from an MIT-licensed
TypeScript deckbuilder template. That legacy code lives under `src/legacy/` (game, round, shop, etc.) and is not part of the main Pegwave play path.

## Distribution

Review license terms before public distribution. Replace or rewrite any
scaffold code you do not have rights to ship.
