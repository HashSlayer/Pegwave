# Pegwave

Plinko synth roguelike — drop voice coins through oscillating pegs; every hit makes sound, keyed and tempo-quantized by the LFO you tune live.

## Quick start

```bash
cd pegwave
bun install
bun run dev
```

Open **http://localhost:3000/**
